### A Modern MERN Stack Social Media Platform

**Shavio** is a full-stack social media platform built with the **MERN Stack** (MongoDB, Express.js, React, and Node.js). It provides users with a modern and intuitive experience for sharing photos, creating posts, interacting with content, and connecting with others in a secure and responsive environment.

Designed with scalability, clean architecture, and user experience in mind, Shavio features secure JWT authentication, cloud-based image storage, responsive design, and a sleek interface inspired by today's leading social media platforms.

## ✨ Key Features

- 🔐 Secure User Authentication (JWT)
- 👤 User Registration & Login
- 🏠 Personalized Home Feed

- 📝 Create, Edit & Delete Posts

- 👥 User Profiles
- 📱 Fully Responsive Design
- ⚡ Fast RESTful API

## 🎯 Project Goal

Shavio was developed to demonstrate modern full-stack web development practices using the MERN stack. The project focuses on creating a scalable, secure, and visually appealing social media platform while implementing real-world features such as authentication, media uploads, user interaction, and responsive UI design.
